nimbleSHOP

Payments

nimbleSHOP has built in support for Authorize.net .

Chapters

  1. Using pre-configured sandbox account
  2. Getting your test account

1 Using pre-configured sandbox account

nimbleSHOP comes with a pre-configured account which can be used to test your application. Be sure to change the values before you go live.

2 Getting your test account

Authorize.net provides an instant test account . Use this test account while your application for merchant account is still being processed.
