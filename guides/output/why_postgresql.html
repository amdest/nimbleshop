Chapters

  1. Heroku makes life easier

1 Heroku makes life easier

For good or the bad nimbleSHOP has totally embraced heroku. Since heroku has buitin support for postgresql, nimbleSHOP decided to use postgresql. So far no code is written specifically for postgresql so nimbleSHOP should work fine with MySQL and other databases. However note that all the tests are run only for postgresql.
