h3. Using pre-configured sandbox account

nimbleSHOP comes with a pre-configured account which can be used to test your application. Be sure to change the values before you go live.

h3. Getting your test account

Authorize.net provides an instant "test account":https://developer.authorize.net/testaccount . Use this test account while your application for merchant account is still being processed.
